# Gean Metrics Package

Performance monitoring and metrics collection for the Gean Lean Ethereum client.

## Purpose

This package provides comprehensive metrics collection and monitoring, including:
- Performance benchmarking
- Consensus operation metrics
- Network statistics
- Resource utilization monitoring
- Export to monitoring systems

## Key Components

- **collector**: Metrics collection framework
- **consensus**: Consensus-specific metrics
- **network**: Network performance metrics
- **system**: System resource monitoring
- **export**: Metrics export formats

## Lean Consensus Metrics

- Signature verification performance
- Block propagation times
- Final latency measurements
- Validator participation rates
- P2P network health
- Resource utilization tracking

## Export Formats

- Prometheus metrics format
- JSON metrics API
- Grafana dashboard support
- Custom metric endpoints
