# Gean Tests Package

Test suite for the Gean Lean Ethereum client.

## Purpose

This package contains comprehensive tests for all Gean components, including:
- Unit tests for individual packages
- Integration tests for component interaction
- Consensus protocol compliance tests
- Performance benchmarks
- Interoperability tests

## Test Categories

- **unit**: Package-level unit tests
- **integration**: Cross-package integration tests
- **consensus**: Lean Consensus protocol tests
- **interop**: Multi-client interoperability tests
- **benchmark**: Performance and load testing

## Lean Consensus Testing

- leanSpec compliance tests
- Post-quantum signature verification
- 3-slot finality protocol tests
- Network protocol compliance
- State transition validation

## Test Data

- Test vectors from leanSpec
- Mock implementations for testing
- Performance benchmark data
- Interop test scenarios
