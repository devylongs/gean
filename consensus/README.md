# Gean Consensus Package

Core consensus logic for the Gean Lean Ethereum client.

## Purpose

This package implements the Lean Consensus protocol, including:
- 3-slot finality (3SF) implementation
- Block attestation and proposal logic
- Validator state management
- Fork choice rules
- Consensus state transitions

## Key Components

- **consensus**: Main consensus engine
- **validator**: Validator operations and state
- **finality**: 3-slot finality protocol
- **forkchoice**: Fork choice implementation
- **state**: Consensus state management

## Lean Consensus Features

- Post-quantum signature integration
- Fast finality (seconds)
- Attester-proposer separation support
- Lean governance mechanisms
